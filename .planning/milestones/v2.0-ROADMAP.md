## Milestone Summary

**Key Decisions:**
- FastAPI + uvicorn chosen over Flask because deepgram-sdk requires asyncio
- socketio.ASGIApp wraps FastAPI — pointing uvicorn at fastapi_app causes 404s
- AsyncServer has no test_client() — tests require real UvicornTestServer + socketio.AsyncClient
- SDK callbacks must be async def with **kwargs — sync/missing-kwargs silently never fire
- Audio timeslice 250ms (not 1000ms) — larger chunks drop final words on Stop
- stream_started emitted immediately on WS connect, not after Metadata (Metadata non-deterministic, 10s+ delay)
- Deepgram boolean params must be lowercase strings ("true"/"false"), not Python bools

**Issues Resolved:**
- gevent monkey.patch_all() corrupts asyncio event loop — deleted as first edit
- pytest-asyncio pinned to <1.0 — v1.3.0 double-instantiates session fixtures
- Test server port changed 8765→9765 — port occupied by node process in container

**Post-ship improvements (same session):**
- Error surfacing: stream_error now emitted to client with cleaned message (headers/token stripped)
- Race condition fix: session popped before stream_error so immediate retry isn't blocked
- Frontend cleanup: MediaRecorder stopped on stream_error so next Start works cleanly
- Redact options: tested all values against streaming API, removed invalid (address/phone/email), added granular PII options
- Footer links, updated README with screenshot
